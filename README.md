# Abey Dental Clinic App

## Overview
The **Abey Dental Clinic App** is designed to streamline dental clinic operations, enhance patient experience, and manage appointments efficiently. This mobile application provides users with a platform to book appointments, view dentist profiles, and receive reminders for upcoming visits. For clinic administrators, it offers tools to manage patient records, schedules, and treatment histories.

---

## Features

### For Patients:
- **Appointment Booking:** Schedule dental appointments with ease.
- **Dentist Profiles:** View qualifications, expertise, and availability of dentists.
- **Treatment History:** Access past appointment and treatment records.
- **Notifications:** Receive reminders for upcoming appointments and follow-ups.
- **Feedback System:** Rate and review clinic services.

### For Administrators:
- **Patient Management:** Add, update, and delete patient records.
- **Appointment Management:** Track and organize daily schedules.
- **Billing and Invoicing:** Generate bills and invoices for treatments.
- **Treatment Plans:** Manage and document patient treatment plans.
- **Notifications:** Receive reminders for upcoming appointments and follow-ups.
- **Feedback System:** Review clinic feedback from patients.

## Technologies Used

- **Backend:** Firebase
- **Database:** Firebase Firestore
- **Authentication:** Firebase Authentication
- **Cloud Storage:** Firebase Storage for images and files
- **Notifications:** Firebase Cloud Messaging (FCM)

## Installation Guide

### Prerequisites:
- Java Development Kit (JDK) installed
- Android Studio installed
- Firebase account set up with a project for the app
- Mobile device or emulator for testing
- SDK Mininum 24 

### Steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/username/abey-dental-clinic.git
   ```

2. Open the project in Android Studio:
   - Launch Android Studio.
   - Select **Open an Existing Project** and navigate to the cloned directory.

3. Set up Firebase:
   - Add the `google-services.json` file to the `app/` directory.

4. Sync the project:
   - Click on **Sync Now** in Android Studio to download dependencies.

5. Run the app:
   - Connect a physical device or start an emulator.
   - Click on the **Run** button in Android Studio.

## Usage Instructions

### Patient:
1. Sign up using email or social media accounts.
2. Browse dentist profiles and book appointments.
3. View treatment history and receive reminders.

### Administrator:
1. Log in with admin credentials including the unique code from administrator.
2. Access the dashboard to manage appointments and patients.
